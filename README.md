# Hexemy: AI-Powered Hex-to-C Decompiler
##### ***[STILL IN PROGRESS]***

---

An AI-powered decompiler that uses **PyTorch** to translate raw **hexadecimal machine code** directly back into human-readable **C source code**.

---

### Project Overview
**Hexemy** treats reverse engineering as a neural machine translation problem. Utilizing a compact **Small Language Model (SLM)** built on a **Transformer** architecture, it translates compiled hexadecimal sequences into structured C syntax.
